Server application should be launched as administrator because the application adds exceptions to firewall (they are removed after closing). File install.bat is simple script that adds a shortcut to your desktop which should be launched in administrator mode.
INSTRUCTIONS:
1. Make sure that ServerApp.jar and install.bat are in the same directory.
2. Run install.bat (double-click). This will add a shortcut to your desktop.
3. To launch ServerApp run it from the shortcut on your desktop in administrator mode.

REQUIREMENTS:
1. Windows 7/8/8.1/10
2. JRE 1.8
3. MS PowerPoint
4. Wi-fi connection.
